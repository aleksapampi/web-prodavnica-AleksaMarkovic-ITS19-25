# Mini web prodavnica 

## Naziv prodavnice 
Niš Coffee Club 
 
## Tema prodavnice 
Coffee lifestyle – šolje, majice i aksesoari inspirisani kulturom ispijanja kafe 
 
## Opis projekta 
Ovaj repozitorijum sadrži osnovne elemente mini web prodavnice brenda Niš Coffee Club, koji je fokusiran na prodaju šolja, garderobe i aksesoara sa coffee tematikom. Projekat je pripremljen za kasniju realizaciju u WordPress/WooCommerce okruženju.
